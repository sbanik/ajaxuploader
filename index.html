ajaxuploader

A file upload plugin which uploads a file and shows the progress in a HTML5 progress bar using the default HTML5 tag. Every file to be uploaded has its own progress bar which displays the progress of every file being uploaded.

The plugin exploits the xhr option of the $.ajax() function of the jQuery library. Hence it is possible to attach a separate progress bar for each file upload. Besides the files are uploaded one by one by looping through the file array extracted by javascript.It is seen to be compatible with Mozilla Firefox v23.0.1 and in Chrome Version 29.0.1547.62
